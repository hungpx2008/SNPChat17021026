# Plan 02: Backend Structural Splits — docling_service + chat_tasks + task dispatcher

## Goal
Split the two largest backend files (`docling_service.py` at 821 LOC, `chat_tasks.py` at 1129 LOC) into focused modules, and extract a task dispatch helper to DRY up Celery `.delay()` calls.

## must_haves
- `backend/src/services/docling/` package exists with 5+ modules, each under 250 LOC
- `docling_service.py` becomes a thin re-export shim (< 30 LOC) — `process_document_deep()` still importable from `src.services.docling_service`
- `backend/src/worker/rag/` package exists with context, search_helpers, synthesis modules
- `chat_tasks.py` has Celery task functions and imports from `rag/` submodules — each task function is < 80 LOC
- All Celery task `name=` strings remain unchanged (`"src.worker.tasks.*"`)
- `backend/src/services/task_dispatcher.py` exists with typed dispatch functions
- `tasks.py` re-export shim continues to work
- All existing imports from `src.worker.tasks` and `src.worker.chat_tasks` still resolve

## threat_model
- **Celery task name stability**: Task decorators use `name="src.worker.tasks.X"` — if these change, running workers fail to find tasks. All names MUST remain identical.
- **Import compatibility**: Many files import from `src.worker.tasks` and `src.worker.chat_tasks`. The re-export shim and the `__init__.py` must preserve all public names.
- **Docling inner classes**: `AdaptiveTableSerializer` inherits from Docling's `BaseTableSerializer` and references `self` from the outer class. Extracting requires passing config explicitly — test with a real document parse.
- **Circular imports**: Splitting `chat_tasks.py` into submodules risk circular imports if modules cross-reference. Mitigation: helper modules MUST NOT import from `chat_tasks.py` — data flows downward only.

```python
"""Typed helpers for dispatching Celery background tasks.

Centralizes the `.delay(...)` calls that were previously scattered across
api/chat.py and services/chat_service.py, eliminating repeated unpacking
of session_id / message_id / content / role / user_id / department.
"""
from __future__ import annotations

import logging
from uuid import UUID

logger = logging.getLogger(__name__)


def dispatch_chat_embed(
    *,
    session_id: str | UUID,
    message_id: str | UUID,
    content: str,
    role: str,
    user_id: str | None,
    department: str | None,
) -> None:
    """Enqueue chunk+embed for a chat message."""
    from src.worker.tasks import process_chat_response

    process_chat_response.delay(
        session_id=str(session_id),
        message_id=str(message_id),
        content=content,
        role=role,
        user_id=user_id,
        department=department,
    )


def dispatch_rag_search(
    *,
    question: str,
    session_id: str | UUID,
    user_id: str | None,
    department: str | None,
    target_message_id: str | UUID,
    mode: str = "rag",
) -> None:
    """Enqueue RAG document search for a user question."""
    from src.worker.tasks import rag_document_search

    rag_document_search.delay(
        question=question,
        session_id=str(session_id),
        user_id=user_id,
        department=department,
        target_message_id=str(target_message_id),
    )


def dispatch_sql_query(
    *,
    question: str,
    session_id: str | UUID,
    target_message_id: str | UUID,
    user_id: str | None = None,
    department: str | None = None,
) -> None:
    """Enqueue SQL query generation via Vanna."""
    from src.worker.tasks import run_sql_query

    run_sql_query.delay(
        question=question,
        session_id=str(session_id),
        target_message_id=str(target_message_id),
        user_id=user_id,
        department=department,
    )


def dispatch_summary_check(*, session_id: str | UUID) -> None:
    """Enqueue session history summarization check."""
    from src.worker.tasks import summarize_session_history

    summarize_session_history.delay(session_id=str(session_id))


def dispatch_store_memory(
    *,
    session_id: str | UUID,
    content: str,
    user_id: str | None,
    role: str = "user",
) -> None:
    """Enqueue long-term memory storage via Mem0."""
    from src.worker.tasks import store_memory

    store_memory.delay(
        session_id=str(session_id),
        content=content,
        user_id=user_id,
        role=role,
    )
```
