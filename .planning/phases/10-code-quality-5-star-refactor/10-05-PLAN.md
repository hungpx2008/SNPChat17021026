# Plan 05: Tests — Unit Tests for Sanitizers, RAG Helpers, and Security Tightening

## Goal
Write comprehensive unit tests for all pure-function sanitization and RAG helper logic (30+ test cases), and verify the CORS tightening from Plan 01 is complete. This wave provides the safety net that validates the refactoring done in Waves 1-4.

## must_haves
- `backend/tests/test_sanitization.py` exists with tests for `_sanitize_generated_answer`, `_clean_snippet_text`, `_strip_markdown_tables`, `_split_sentences_vi`, `_format_citations_footer`, `_build_fallback_answer`
- `backend/tests/test_rag_helpers.py` exists with tests for `_build_qdrant_filter`, `_resolve_parent_content`
- `frontend/src/__tests__/content-sanitizers.test.ts` exists with tests for all 6 exported functions
- All tests pass: `pytest` (backend) and `npx jest` (frontend)
- CORS middleware uses explicit method/header lists (verified in Plan 01, re-confirmed here)

## threat_model
- **Test isolation**: Backend pure functions (`_sanitize_generated_answer`, etc.) must be testable without Docker/Redis/Qdrant. They are pure functions with no I/O — import directly.
- **CORS security verification**: After all changes, verify that `allow_methods=["*"]` no longer appears anywhere in the codebase. A wildcard method allowance could permit unexpected HTTP methods.
- **Test regression safety**: These tests serve as regression guards for the refactoring. If any test fails during CI, it means the refactoring broke behavior.

        result = _sanitize_generated_answer(text)
        assert "Lưu ý" not in result

    """Tests for _clean_snippet_text."""

        assert _clean_snippet_text("") == ""

    def test_removes_extra_whitespace(self):
        result = _clean_snippet_text("  hello   world  ")
        assert "  " not in result.strip()

    def test_preserves_vietnamese_chars(self):
        text = "Cảng container quốc tế Cát Lái"
        assert _clean_snippet_text(text) == text

    """Tests for _strip_markdown_tables."""

    def test_preserves_non_table_content(self):

    """Tests for _split_sentences_vi (Vietnamese sentence splitting)."""

    def test_empty_string(self):
        assert _split_sentences_vi("") == []

    def test_single_sentence(self):
        result = _split_sentences_vi("Xin chào.")
        assert len(result) >= 1

    def test_multiple_sentences(self):
        text = "Câu một. Câu hai. Câu ba."
        result = _split_sentences_vi(text)
        assert len(result) >= 2

    def test_preserves_abbreviations(self):
        """Common Vietnamese abbreviations should not cause false splits."""
        text = "TP. Hồ Chí Minh có cảng lớn."
        result = _split_sentences_vi(text)

        assert result.count("doc.pdf") <= 2  # At most one citation entry

import {
  sanitizeBotContent,
  stripCodeBlocks,
  stripPythonTracebacks,
  deduplicateCitations,
  cleanSystemNotes,
  stripMalformedTables,
} from '@/lib/content-sanitizers';

describe('stripCodeBlocks', () => {
  it('removes SQL code blocks', () => {
    const input = 'Result: ```sql\nSELECT * FROM users\n``` end';
    expect(stripCodeBlocks(input)).not.toContain('```');
  });

  it('removes Python code blocks', () => {
    const input = 'Result: ```python\nprint("hello")\n``` end';
    expect(stripCodeBlocks(input)).not.toContain('```');
  });

  it('preserves non-code content', () => {
    const input = 'Normal text without code blocks';
    expect(stripCodeBlocks(input)).toBe(input);
  });

  it('handles empty string', () => {
    expect(stripCodeBlocks('')).toBe('');
  });
});

describe('stripPythonTracebacks', () => {
  it('removes traceback text', () => {
    const input = 'Answer\nTraceback (most recent call last)\n  File "test.py"\nValueError: bad';
    expect(stripPythonTracebacks(input)).not.toContain('Traceback');
  });

  it('preserves normal text', () => {
    const input = 'Just a normal response';
    expect(stripPythonTracebacks(input)).toBe(input);
  });
});

describe('deduplicateCitations', () => {
  it('removes duplicate citations', () => {
    const input = '[nguồn: doc.pdf, trang 1] text [nguồn: doc.pdf, trang 1]';
    const result = deduplicateCitations(input);
    const matches = result.match(/\[nguồn:/g);
    expect(matches?.length ?? 0).toBeLessThanOrEqual(1);
  });

  it('keeps unique citations', () => {
    const input = '[nguồn: doc1.pdf] text [nguồn: doc2.pdf]';
    const result = deduplicateCitations(input);
    expect(result).toContain('doc1.pdf');
    expect(result).toContain('doc2.pdf');
  });
});

describe('cleanSystemNotes', () => {
  it('removes system advisory notes', () => {
    const input = 'Answer\nLưu ý: Đây là thông tin thô từ tài liệu gốc';
    expect(cleanSystemNotes(input)).not.toContain('Lưu ý');
  });

  it('preserves normal Vietnamese text', () => {
    const input = 'Cảng Tân Cảng Sài Gòn hoạt động 24/7.';
    expect(cleanSystemNotes(input)).toBe(input);
  });
});

describe('stripMalformedTables', () => {
  it('removes orphan pipe lines', () => {
    const input = 'Text\n|---|---|\nMore text';
    expect(stripMalformedTables(input)).not.toContain('|---|');
  });
});

describe('sanitizeBotContent (full pipeline)', () => {
  it('handles empty string', () => {
    expect(sanitizeBotContent('')).toBe('');
  });

  it('applies all sanitizers', () => {
    const input = [
      'Answer ```sql\nSELECT 1\n```',
      'Traceback (most recent call last)\nValueError: bad',
      '[nguồn: doc.pdf] [nguồn: doc.pdf]',
      'Lưu ý: Đây là thông tin thô từ tài liệu gốc',
      '|---|---|',
    ].join('\n');
    const result = sanitizeBotContent(input);
    expect(result).not.toContain('```');
    expect(result).not.toContain('Traceback');
    expect(result).not.toContain('Lưu ý');
    expect(result).not.toContain('|---|');
  });

  it('preserves legitimate content', () => {
    const input = 'Tân Cảng Sài Gòn xử lý hàng triệu TEU mỗi năm.';
    expect(sanitizeBotContent(input)).toBe(input);
  });
});
